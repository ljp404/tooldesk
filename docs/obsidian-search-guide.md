# Obsidian 搜索功能使用指南

## 功能说明

Obsidian 搜索是一个本地文件库搜索工具，可以快速搜索和打开 Obsidian 笔记、Markdown 文件等本地文档。

## 配置步骤

### 1. 打开设置

点击主程序右上角的设置图标，打开设置面板。

### 2. 进入本地库配置

在左侧导航栏中，点击"本地库"选项卡。

### 3. 添加库配置

点击"添加第一个库"或"+ 添加库"按钮，填写以下信息：

- **关键字*** (必填)
  - 例如：`ob`
  - 用于快捷启动器中快速进入该库搜索
  - 最多 10 个字符

- **名称*** (必填)
  - 例如：`Obsidian 笔记`
  - 库的显示名称

- **路径*** (必填)
  - 例如：`C:\Users\Admin\Documents\Obsidian\MyVault`
  - 本地文件夹的完整路径

- **文件类型**
  - 例如：`.md, .txt`
  - 多个扩展名用逗号分隔
  - 默认：`.md`

- **打开方式**
  - 默认程序
  - Typora（需要配置 Typora 路径）
  - VS Code
  - Obsidian（需要配置库名）

- **Typora 路径**（当选择 Typora 时显示）
  - 例如：`C:\Program Files\Typora\Typora.exe`

- **Obsidian 库名**（当选择 Obsidian 时显示）
  - 例如：`MyVault`
  - 用于 Obsidian URI 协议

### 4. 保存配置

点击"保存"按钮完成配置。

## 使用方式

### 方式一：主程序搜索

1. 在主程序顶部搜索框中输入关键字
2. 可以搜索：`obsidian`、`ob`、`笔记`、`markdown` 等
3. 点击"Obsidian 搜索"工具打开

### 方式二：快捷启动器

1. 按下 `Alt + Space` 打开快捷启动器
2. 输入配置的关键字（如 `ob`）
3. 按 `Enter` 进入库搜索模式
4. 输入搜索关键字
5. 使用 `↑` `↓` 键选择文件
6. 按 `Enter` 打开文件

## 搜索功能

- **文件名搜索**：根据文件名匹配
- **内容搜索**：搜索文件内容（可在设置中关闭）
- **显示匹配行**：显示匹配内容所在的行号和上下文
- **跳转到行**：VS Code 直接支持；Obsidian 需安装社区插件 [Advanced URI](https://github.com/Vinzent03/obsidian-advanced-uri)，并正确填写库名

## 键盘快捷键

- `↑` `↓`：选择文件
- `Enter`：打开选中的文件
- `Esc`：清空搜索或返回库列表

## 注意事项

1. 确保配置的路径存在且可访问
2. 如果选择 Typora 或 Obsidian，需要确保程序已安装
3. Obsidian 需已运行；按行打开使用 Advanced URI（`obsidian://adv-uri`），库名须与 Obsidian 左侧显示的 vault 名称一致
4. 大型库可能需要较长的扫描时间，可以在设置中调整最大扫描文件数

## 高级设置

在"本地库"设置中，还可以配置：

- **最大结果数**：每次搜索返回的最大结果数（默认 50）
- **搜索内容**：是否搜索文件内容（默认开启）
- **最大扫描文件数**：扫描文件的最大数量（默认 500）
