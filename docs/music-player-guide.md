# 音乐播放器使用指南

## 功能概述

音乐播放器组件支持两种音乐源：
1. **本地音乐**：扫描本地文件夹中的音乐文件
2. **阿里云盘**：通过扫码登录访问云盘中的音乐文件

## 本地音乐

### 支持的格式
- MP3
- FLAC
- M4A
- WAV
- AAC
- OGG
- WMA

### 使用步骤
1. 点击"本地音乐"标签
2. 点击"选择音乐文件夹"按钮
3. 选择包含音乐文件的文件夹
4. 等待扫描完成
5. 双击音乐或点击播放按钮开始播放

### 功能特性
- 自动扫描文件夹及子文件夹
- 支持搜索歌曲、艺术家或专辑
- 显示音乐列表和播放状态

## 阿里云盘音乐

### 使用步骤
1. 点击"阿里云盘"标签
2. 点击"生成登录二维码"按钮
3. 使用阿里云盘 App 扫描二维码
4. 在手机上确认登录
5. 登录成功后自动加载云盘根目录
6. 浏览文件夹，找到音乐文件
7. 双击音乐或点击播放按钮开始播放

### 功能特性
- 扫码登录，无需手动输入账号密码
- 自动管理访问令牌
- 支持文件夹浏览
- 支持搜索音乐
- 自动获取播放链接

### 注意事项
- 首次登录需要扫码授权
- 登录状态会保存，下次打开自动登录
- 如果遇到播放问题，可能是令牌过期，会自动刷新
- 点击"退出登录"可以清除登录状态

## 播放控制

### 播放器功能
- **播放/暂停**：点击中间的播放按钮
- **上一曲/下一曲**：点击左右箭头按钮
- **播放模式**：
  - 顺序播放：按列表顺序播放
  - 单曲循环：重复播放当前歌曲
  - 随机播放：随机选择歌曲播放
- **进度控制**：拖动进度条跳转到指定位置
- **音量控制**：拖动音量滑块调整音量

### 快捷操作
- 双击音乐项：立即播放该音乐
- 单击播放按钮：播放/暂停该音乐

## 技术说明

### 本地音乐实现
- 使用 Tauri 原生命令扫描本地文件
- 支持递归扫描子文件夹
- 使用 HTML5 Audio API 播放音乐

### 阿里云盘实现
- 使用阿里云盘官方 API
- 扫码登录流程：
  1. 生成二维码
  2. 轮询检查扫码状态
  3. 获取 refresh_token
  4. 使用 refresh_token 获取 access_token
- 自动管理令牌刷新
- 获取临时下载链接用于播放

### 数据存储
- 阿里云盘的 refresh_token 存储在本地
- 位置：`%APPDATA%/tooldesk/aliyun-token.json`
- 退出登录会清除该文件

## 常见问题

### 本地音乐无法播放
- 检查文件格式是否支持
- 检查文件是否损坏
- 尝试重新选择文件夹

### 阿里云盘登录失败
- 检查网络连接
- 确保使用最新版阿里云盘 App
- 尝试刷新二维码重新扫码

### 云盘音乐播放卡顿
- 检查网络速度
- 云盘文件下载速度受网络影响
- 建议使用稳定的网络环境

### 找不到音乐文件
- 本地音乐：确保文件扩展名正确
- 云盘音乐：确保文件在云盘中存在且未被删除
